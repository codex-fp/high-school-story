# Design Topics To Revisit

## Purpose

This document tracks design topics that are intentionally deferred.

These are not forgotten tasks. They are areas that need a later decision, usually because the direction is still unclear or needs more thought before being locked into the design.

## Deferred Topics

### Narrative teacher archetypes

Status:

- deferred for later discussion

Reason:

- teachers now have only limited mechanical importance in the MVP
- teacher archetypes are considered a later narrative polish layer rather than an MVP-critical requirement

What is already settled:

- teachers should matter mainly through tone, framing, and character feel
- teachers should not drive major mechanical differences between classes in the MVP

What is still open:

- whether the game should use a formal set of recurring teacher archetypes at all
- how many teacher personality types are useful for later narrative content
- whether teacher archetypes should live in design docs or later narrative documentation

When to revisit:

- revisit this topic when expanding school flavor, narrative depth, or post-MVP classroom personality work

### Preferences influence on school subjects

Status:

- deferred for later discussion

Reason:

- the direction is not yet decided strongly enough to lock into the design
- the topic needs more thought before we define how preference tags affect school subjects

What is already settled:

- attributes are the main driver of subject performance and progress
- `composure` should act more like a shared modifier, especially under stress, than like a subject-specific weighted stat
- subject-to-attribute mapping should be explicit in design and use simple weights

What is still open:

- whether preferences should influence school subjects at all
- if they do, whether they should affect only `stress` and `mood`, or something else too
- how strong that influence should be
- whether every subject should connect to preferences in the same way

When to revisit:

- revisit this topic when we explicitly return to subject comfort, school identity, or preference-system integration
