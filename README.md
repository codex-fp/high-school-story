# High School Story

Godot 4.7 / C# high-school life simulation project with gameplay rules kept in clean .NET boundaries.

## Development commands

Run these from the repository root:

```powershell
dotnet restore "High School Story.sln"
dotnet build "High School Story.sln"
dotnet test
```

For the complete tool, test, and Godot smoke command reference, see [the implementation guide](docs/implementation.md).
